Overview

Financial AI Analyzer is a self-hosted bookkeeping assistant that combines document management with AI-powered analysis. It connects to your Paperless-ngx document library, imports transactions from Gmail receipts, PayPal, Venmo, and bank CSV files, and lets you query everything through a conversational AI interface.

📄 Document Analysis

Pull documents from Paperless-ngx and let AI extract amounts, vendors, dates, and tax categories automatically.

💲 Transaction Import

Import transactions from Gmail, PayPal, Venmo, and any bank's CSV export in a single click.

🤖 AI Chat

Ask questions like "What did I spend on software in 2024?" and get instant answers from your data.

📈 Reports & Exports

Generate profit & loss summaries, export to CSV, JSON, IIF (QuickBooks), OFX, TXF, or PDF.

🏠 Multi-Entity

Track multiple entities (personal, business, farm) in a single installation with per-entity filtering.

🔒 Self-Hosted

All data stays on your server. No third-party cloud sync or SaaS subscriptions required.

System Requirements

  • Docker / Docker Compose on the host server
  • Paperless-ngx instance (for document analysis features)
  • An Anthropic API key (Claude) or OpenAI API key for AI features
  • A modern web browser — Chrome, Firefox, Safari, or Edge

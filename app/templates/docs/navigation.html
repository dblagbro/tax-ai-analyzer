Navigation

The interface is a single-page application with a fixed left sidebar and a top bar.

Top Bar

  • Entity selector — filter all views to a specific entity (Personal, VoIPGuru, Martinfeld Ranch, etc.).
  • Tax Year selector — filter all views to a specific tax year.
  • User menu — access settings and log out.

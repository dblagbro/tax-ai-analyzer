URL Importer

The URL importer downloads a CSV or JSON file from a URL on a recurring schedule — useful for integrations that expose a downloadable export endpoint.

  1. Go to Import → URL.
  2. Enter the URL, file format (CSV/JSON), entity, and tax year.
  3. Set the polling interval (in minutes).
  4. Click Add URL Poller. The background scheduler will fetch the URL at the specified interval.

Authentication

If the URL requires HTTP Basic authentication, enter the credentials in the Username and Password fields. For Bearer tokens, enter the token in the Password field and leave Username empty.

Logging In

Navigate to the app URL (e.g., https://yourserver.example.com/tax-ai-analyzer). You will be redirected to the login page if you are not already authenticated.

  1. Enter your Username and Password.
  2. Check Remember me if you want to stay logged in across browser sessions.
  3. Click Sign In.
Default credentials The default admin account is username admin with password admin123. Change this immediately after first login via Settings → Users.
Session expiry Sessions expire after 24 hours by default. With "Remember me" enabled, the session persists for 30 days via a secure cookie.

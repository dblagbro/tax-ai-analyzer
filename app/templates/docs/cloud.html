Cloud Storage Import

The cloud adapter connects to Google Drive, Dropbox, or Amazon S3 to scan for CSV and PDF files and import them automatically.

Supported Providers

ProviderCredential required
Google DriveService account JSON or OAuth token
DropboxApp access token
Amazon S3Access key ID + secret + bucket name

Configuration

  1. Go to Import → Cloud.
  2. Select a provider and enter the required credentials.
  3. Specify the folder or bucket path to scan.
  4. Click Connect & Scan. A background job will fetch matching files.
Optional feature Cloud adapters require optional Python packages (google-api-python-client, dropbox, boto3). If not installed, the Cloud tab will display an installation notice.

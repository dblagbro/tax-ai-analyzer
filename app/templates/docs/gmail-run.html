Running a Gmail Import

Once credentials and a token are configured, running an import is one click.

  1. Go to Import → Gmail.
  2. Select the Entity and Tax Year to associate with imported transactions.
  3. Click Start Gmail Import.
  4. The job runs in the background. Watch the progress in Import → Import Jobs.

What gets imported

The importer searches for emails matching these keywords: receipt invoice statement payment billing 1099 W-2 tax order. For each matching email, it:

  • Extracts the subject, sender, date, and body.
  • Downloads PDF/image attachments and sends them to Paperless-ngx for OCR and storage.
  • Uses AI to extract vendor name, amount, category, and tax year from the content.
  • Creates a transaction record with the extracted data.
Years covered By default the importer covers 2022–2025. Change this in Settings → Gmail Years, or set the GMAIL_YEARS environment variable.

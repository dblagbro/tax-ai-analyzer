Troubleshooting

Login issues

ProblemSolution
"Invalid username or password"Check username spelling. Default credentials: admin / admin123.
Redirected to login on every pageClear browser cookies. Ensure the server is accessible via HTTPS (required for secure cookies in production).
Account locked / inactiveAn admin must reactivate the account in the Users tab.

Import issues

ProblemSolution
Gmail import fails immediatelyCheck that credentials.json and the OAuth token are both present in Settings → Gmail status. Re-run Gmail Setup if the token is expired.
CSV import shows 0 recordsCheck that the file has the expected column names. See CSV Column Reference above. Try opening the file in a text editor to inspect the headers.
Import job stuck at "running"Check the container logs: docker logs tax-ai-analyzer. Restart the container if the job has not progressed in 10+ minutes.

AI / LLM issues

ProblemSolution
Chat returns "API key not configured"Go to Settings and enter your Anthropic or OpenAI API key. Click Test LLM to verify.
AI analysis is slowLarge documents take longer. Check container resource limits in docker-compose.yml. Reduce parallel analysis workers if memory is constrained.
AI gives wrong categoriesYou can manually override categories on any transaction. The AI uses description and vendor — adding more context in the Notes field improves future suggestions.

Paperless-ngx connection

ProblemSolution
Documents tab shows 0 documentsVerify Paperless URL and API token in Settings. Click Test Paperless. Ensure Paperless is running: docker ps | grep paperless.
Documents show "error" badgeCheck container logs for the specific document ID. Common cause: OCR extracted no text and Vision AI fallback is not enabled.

Checking logs

# View real-time logs for the analyzer
docker logs -f tax-ai-analyzer

# View last 100 lines
docker logs --tail 100 tax-ai-analyzer

# View Paperless-ngx logs
docker logs -f tax-paperless-web
Activity feed The Dashboard tab shows a real-time activity feed of recent operations. This is often the fastest way to see what happened during an import or analysis run without needing to check logs.

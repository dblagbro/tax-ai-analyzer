AI Document Analysis

The AI analyzer polls Paperless-ngx for new documents and automatically extracts financial data. You can also trigger analysis manually.

Triggering analysis manually

  1. Go to the Documents tab.
  2. Click Run AI Analysis (top right).
  3. The analyzer will process any unanalyzed documents. Progress is shown in the activity feed.

Vision AI fallback

When the analyzer detects poor OCR quality (e.g., scanned images, handwritten forms), it automatically falls back to sending document images directly to Claude's Vision API. Documents processed this way are tagged aianomaly:vision_ai_extracted in Paperless-ngx.

Paperless-ngx connection AI analysis requires a running Paperless-ngx instance. Configure the URL and API token in Settings → Paperless.

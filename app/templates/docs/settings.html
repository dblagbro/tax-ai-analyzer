Settings

The Settings tab configures connections, API keys, and application behavior.

SettingDescription
LLM Provideranthropic or openai
LLM API KeyYour Anthropic or OpenAI API key
LLM ModelModel name (e.g. claude-sonnet-4-6)
Paperless URLBase URL of your Paperless-ngx instance
Paperless TokenAPI token from Paperless-ngx admin panel
Gmail YearsComma-separated list of years to search
Export PathServer path where exports are saved
Poll IntervalHow often (seconds) to poll Paperless for new documents

Testing connections

After entering settings, use the Test LLM and Test Paperless buttons to verify connectivity before saving.

API key security API keys are stored in the SQLite database on your server. Use HTTPS and keep your server firewall properly configured to prevent unauthorized access.

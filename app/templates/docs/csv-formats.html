CSV Column Reference

When the auto-detector cannot identify your CSV format, ensure your file has columns matching these names (case-insensitive).

Generic bank CSV

ColumnRequiredDescription
dateYesTransaction date (YYYY-MM-DD, MM/DD/YYYY, etc.)
description or memoYesTransaction description
amountYesPositive for deposits, negative for withdrawals
debit / creditAltSeparate debit and credit columns (either/or with amount)
categoryNoPre-assigned category (will be mapped or overridden by AI)
reference or check_numberNoReference number for dedup

PayPal CSV (standard export)

ColumnNotes
DateMM/DD/YYYY format
NameCounterparty name
TypeTransaction type (Payment Sent, etc.)
StatusCompleted transactions are imported; others skipped
GrossAmount with currency symbol (e.g. $-12.50)
FeePayPal fee (added to gross for net amount)
NetUsed if Gross is missing
Transaction IDUsed for dedup

Venmo CSV (standard export)

ColumnNotes
DatetimeISO 8601 format
TypePayment or Charge
StatusComplete transactions only
NotePayment memo (used as description)
FromSender name
ToRecipient name
Amount (total)With + or - sign
IDUsed for dedup

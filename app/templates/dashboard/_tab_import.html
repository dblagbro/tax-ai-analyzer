Log in to US Alliance from your real browser, then use one of these methods:

Method A — Console snippet (no extension needed):
Open DevTools (F12) → Console tab, then click the button below to copy the snippet, paste it in the console, and press Enter:
copy(JSON.stringify(document.cookie.split(';').map(c=>{var p=c.trim().indexOf('=');return{name:c.trim().slice(0,p),value:c.trim().slice(p+1),domain:'account.usalliance.org',path:'/'};}).filter(c=>c.name)))
Then paste (Ctrl+V) into the console and hit Enter, then paste the result into the box below.
Note: this only captures non-HttpOnly cookies. If import still fails, use Method B.
Method B — Cookie-Editor extension (gets all cookies including HttpOnly):
1. Install Cookie-Editor from Chrome Web Store
2. While logged in to US Alliance, click the Cookie-Editor icon in your toolbar
3. Click the Export button at the bottom → JSON is copied to clipboard
4. Paste into the box below

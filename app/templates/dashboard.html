❓ Help & Manual

Welcome to Financial AI Analyzer

Financial AI Analyzer is an AI-powered financial document management and tax preparation tool. It helps you organize documents, track transactions, and prepare for tax season across multiple business entities and years.

Key Features

  • Document Management — Import PDFs from local folders, Gmail, Google Drive, Dropbox, or PayPal. Documents are OCR'd and AI-analyzed automatically.
  • Transaction Tracking — Import bank CSVs and OFX/QFX files. AI categorizes transactions as income, expenses, or deductions.
  • Multi-Entity Support — Manage finances for multiple entities (personal, LLCs, DBAs) with separate books per entity per year.
  • AI Chat — Ask questions about your finances using natural language. Export chat sessions as PDFs.
  • Reports & Exports — Generate PDF reports, CSV exports, OFX/QFX files, TurboTax TXF files, and QuickBooks IIF files.

Getting Started

  • Set up your entities in Admin → Entities
  • Import documents via Tools → Import (Local Folder, Gmail, Google Drive, Dropbox)
  • Review and edit transactions in Financials → Transactions
  • Generate reports in Financials → Reports

Dashboard

The dashboard gives you a financial overview for the selected entity and tax year.

Summary Cards

  • Total Documents — Number of documents imported and analyzed
  • Total Income — Sum of all income transactions for the selected year
  • Total Expenses — Sum of all expense/deduction transactions
  • Net Income / Loss — Income minus expenses

Entity Breakdown

Shows income, expenses, and document counts per entity. Click filter to focus the view on that entity. Click → to jump to the documents for that entity.

Recent Activity

Shows the latest system events including imports, AI analysis completions, and logins.

Status Indicators (top bar)

The colored dots in the top bar indicate system health. Green = healthy, Yellow = degraded, Red = error. Hover over a dot to see details. The entity and year filters in the top bar apply to all views.

Transactions

View, search, filter, and manage all financial transactions across your entities.

Importing Transactions

  • Use Tools → Import to import CSV bank statements or OFX/QFX files
  • Supported formats: Chase, Bank of America, Wells Fargo, Citi, Capital One, PayPal, and generic CSV

Transaction Fields

  • Date — Transaction date
  • Description — Merchant or payee name
  • Amount — Positive = income, Negative = expense
  • Category — AI-assigned category (editable)
  • Entity / Year — Which entity and tax year this belongs to

Editing Transactions

Click any transaction row to edit category, description, entity, or year. Changes are saved immediately.

Filtering

Use the filter bar to search by description, filter by category, entity, year, or date range. Column headers are sortable and resizable.

Documents

All imported PDFs are stored here with AI-extracted metadata.

Document Fields

  • Title — Auto-detected from filename or AI extraction
  • Type — AI-assigned document type (receipt, invoice, bank statement, W-2, 1099, etc.)
  • Vendor / Amount / Date — Extracted by AI from document content
  • Entity / Year — Assigned entity and tax year
  • Tags — Auto-applied tags (e.g., aianomaly:vision_ai_extracted for docs where OCR failed)

Analyze Docs Button

Re-triggers AI analysis on all unanalyzed documents for the current entity/year.

Viewing Documents

Click the document title to open it in Paperless-ngx for full-text search and viewing.

Reports & Exports

Generate financial summaries and tax-ready export files.

Available Exports

  • CSV — Spreadsheet-compatible transaction list
  • PDF Report — Formatted summary suitable for your accountant
  • OFX/QFX — Quicken / QuickBooks Online import format
  • TurboTax TXF — Direct TurboTax import
  • QuickBooks IIF — QuickBooks Desktop import
  • JSON — Machine-readable full export
  • ZIP Bundle — All of the above plus source PDFs in one archive

Generating Reports

Select an entity and year, then click the desired export format button. Large exports may take a few seconds to generate.

Import

Import financial documents and transactions from multiple sources.

Local Folder

Enter a path accessible from the server (e.g., /mnt/s/documents/tax/2023/). The system recursively scans for PDFs, CSVs, and OFX files. Entity is auto-detected from the folder path when possible.

Gmail

Connect your Google account to automatically import financial emails (receipts, invoices, statements) as PDFs. Requires OAuth2 setup — click Connect Gmail Account and follow the authorization steps.

Google Drive

Connect Google Drive to import documents from a specific folder or shared drive.

Dropbox

Connect Dropbox to import from a Dropbox folder path.

PayPal

Connect your PayPal account (Client ID + Secret) to import PayPal transaction history directly.

Import Jobs

All imports run as background jobs. You can view progress and logs in the Import Jobs section. Jobs can be cancelled while running.

AI Chat

Ask questions about your finances using natural language. The AI has access to all your transactions, documents, and entity data.

Example Questions

  • "What were my total business expenses for VoIPGuru in 2023?"
  • "List all income transactions over $1,000 for personal in 2024"
  • "What documents do I have for Martinfeld Ranch?"
  • "Summarize my deductible expenses for 2022"
  • "Which months had the highest expenses last year?"

Chat Sessions

Each conversation is saved as a session. Use the sidebar to switch between sessions, rename them, or delete them.

Editing Messages

Click the pencil icon on any of your messages to edit and resend.

Exporting

Click Export PDF to download the current chat session as a formatted PDF.

Entities

Entities represent the different legal persons or businesses whose finances are tracked. The system supports a hierarchy of entity types.

Entity Types

  • Person — An individual (e.g., Devin Blagbrough). Stores SSN, date of birth, address.
  • LLC — Limited Liability Company. Stores EIN.
  • Corp / S-Corp — Corporation. Stores EIN and Employer ID.
  • DBA — Doing Business As. A trade name under a parent entity. Stores DBA registration number.
  • Trust / Estate — For trust-managed assets.
  • Non-Profit — 501(c)(3) organizations.

Hierarchy

Entities can have parent-child relationships. For example, a DBA is a trade name operated by a Person or LLC. Sub-entities inherit the parent's legal structure for tax purposes.

Adding / Editing Entities

Click + Add Entity to create a new entity. Select the type to see the relevant fields. Click the edit icon on any entity row to modify it.

Merging & Acquiring

Use 🔀 Merge / Acquire to consolidate two entities. All transactions, documents, and records from the source entity are moved to the target. The source entity is archived. This is useful for M&A events, DBA consolidations, or correcting misassigned records.

Settings

System configuration options for administrators.

AI Provider

Configure which AI model is used for document analysis and chat. Supports Anthropic (Claude), OpenAI, and Google Gemini. Enter your API key and select a model.

Paperless Integration

Connect to a Paperless-ngx instance for document storage and OCR. Enter the API base URL and token.

Elasticsearch

Optional vector search integration for semantic document search. Enter the Elasticsearch URL and credentials.

Tax Years

Manage the list of active tax years. Add new years as needed.

Users

Manage user accounts. Admins can add users, reset passwords, and toggle admin status.

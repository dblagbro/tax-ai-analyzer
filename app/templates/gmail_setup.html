Where to get credentials.json

Follow these steps in Google Cloud Console:

  1. Go to console.cloud.google.com and create or select a project
  2. Enable the Gmail API in APIs & Services → Library
  3. Go to OAuth consent screen → select External user type (required for personal @gmail.com accounts)
  4. Fill in app name and your email. Add your Gmail address as a test user
  5. Go to Credentials → Create Credentials → OAuth client ID
  6. Select application type: Desktop app (not Web application)
  7. Download the JSON file and upload it above

Note: Your app will stay in "Testing" mode — that is completely fine. Just make sure to add yourself as a test user on the OAuth consent screen.
